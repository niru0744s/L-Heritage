# L'Héritage Restaurant 

## Project Overview

L'Héritage is a premium, single-page web application designed for a luxury European fine-dining restaurant. Based on an AI-generated UI design from Stitch, it’s built as a fully responsive Next.js application capturing the essence of high-end dining.

The site features a bespoke dark-mode aesthetic utilizing deep zinc tones, custom Work Sans typography, and an elegant gold accent (`#c59f59`). Fluid micro-animations guide visitors through a Hero section, Heritage history block, Curated Menu, Gallery, and an integrated Reservation system.

## Tech Stack

This project is built with modern, performance-first web technologies:

*   **Framework**: Next.js 15 (App Router) with React 19, offering SSR and robust routing.
*   **Language**: TypeScript for end-to-end type safety.
*   **Styling**: Vanilla CSS Modules (`.module.css`) and global variables, ensuring highly maintainable, component-scoped styling without utility-class bloat.
*   **Typography**: `@next/font` with Work Sans and Google Material Symbols for scalable vector icons.
